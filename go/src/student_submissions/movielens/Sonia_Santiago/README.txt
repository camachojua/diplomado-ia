Para ejecutar los programas es necesario contar con el archivo ratings.csv
y también con el archivo movies.csv. Ambos dentro de la carpeta test.


- El programa en Go se ejecuta con el siguiente comando:

   go run mapred_go.go

- El programa en Julia se ejecuta de la siguiente manera:

   julia -t 10 mapred_julia.jl
